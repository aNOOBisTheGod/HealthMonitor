# Health Monitor
# Веб-приложение для мониторинга и анализа показателей здоровья

## Описание проекта

Это веб-приложение предназначено для мониторинга и анализа показателей здоровья пользователя. Приложение позволяет отслеживать такие показатели как пульс, кровяное давление, уровень кислорода в крови, температуру тела и другие параметры.

## Требования по окружению

- Node.js >= 16.0.0
- npm >= 7.0.0
- Modern web browser (Chrome, Firefox, Safari, Edge)

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск локального сервера для разработки
npm run dev

# Лечение кода (ESLint)
npm run lint

# Форматирование кода (Prettier)
npm run format
```

Приложение доступно по адресу `http://localhost:8000`

## Функциональность

### 1. Панель управления
- Отображение текущих показателей здоровья (пульс, давление, SpO2, температура)
- Графики пульса за 24 часа
- Статистика активности
- Автоматическое обновление данных

### 2. Метрики
- Добавление новых измерений различных показателей
- История всех измерений
- Возможность удаления записей
- Добавление заметок к измерениям
- Поиск по датам

### 3. Ежедневные опросы о болевых ощущениях 📋
- 8 типов болевых ощущений (головная боль, мышечная боль, боль в животе и т.д.)
- Оценка интенсивности по шкале 0-4
- Автоматическое сохранение данных о погоде при каждом опросе
- История всех опросов с полной информацией
- **Анализ корреляции болевых ощущений с погодой** (температура, давление, влажность)
- Ежедневное напоминание о прохождении опроса
- Экспорт/импорт опросов вместе с другими данными
- Подробное описание: [SURVEYS_FEATURE.md](SURVEYS_FEATURE.md)

### 4. Анализ
- Статистика пульса (среднее, минимум, максимум, стабильность)
- Уровень активности (прогресс к дневной норме)
- Анализ здоровья дыхательной системы
- Персонализированные рекомендации по улучшению здоровья

### 4. Параметры
- Управление профилем пользователя
- Подсчёт индекса массы тела (BMI)
- Настройка уведомлений
- Экспорт данных в JSON
- Импорт данных из JSON
- Очистка всех данных

### 5. Дизайн
- Адаптивная верстка (мобильные, планшеты, рабочие столы)
- Поддержка светлой и тёмной темы
- Интуитивный интерфейс
- Быстрая загрузка и отклик

## Архитектура

Приложение построено на архитектуре, основанной на объектно-ориентированном программировании (ООП) с использованием классов ES6+.

### Классы

#### `StorageManager`
Управляет сохранением и загрузкой данных из LocalStorage с префиксом для избежания конфликтов.

**Методы:**
- `save(key, data)` - сохранить данные
- `load(key, defaultValue)` - загрузить данные
- `remove(key)` - удалить данные
- `clear()` - очистить все данные

#### `HealthDataAPI`
Интегрирует внешние API для получения данных о здоровье и рекомендаций.

**Методы:**
- `fetchWeatherData(latitude, longitude)` - получить данные о погоде
- `fetchHealthRecommendations()` - получить рекомендации по здоровью
- `generateMockHealthData()` - генерировать тестовые данные

#### `HealthMetrics`
Управляет метриками здоровья и статистикой.

**Методы:**
- `addMetric(type, value, date, notes)` - добавить метрику
- `removeMetric(id)` - удалить метрику
- `getStatistics(type)` - получить статистику по типу
- `getMetricsByType(type)` - получить метрики определённого типа
- `getMetricsInRange(startDate, endDate)` - получить метрики за период

#### `UserProfile`
Управляет профилем пользователя и вычисляет состояние здоровья.

**Методы:**
- `updateProfile(data)` - обновить профиль
- `getProfile()` - получить профиль
- `calculateBMI()` - вычислить индекс массы тела
- `getHealthStatus()` - получить статус здоровья

#### `ChartRenderer`
Рендерит графики и диаграммы с использованием Canvas API.

**Методы:**
- `drawLineChart(canvasId, data, label, color)` - нарисовать линейный график
- `drawBarChart(canvasId, labels, data, color)` - нарисовать столбчатую диаграмму

#### `UIManager`
Управляет пользовательским интерфейсом и взаимодействием.

**Методы:**
- `initializeUI()` - инициализировать интерфейс
- `switchPage(page)` - переключиться на страницу
- `loadDashboard()` - загрузить панель управления
- `loadAnalysis()` - загрузить страницу анализа
- `setupMetricsForm()` - настроить форму добавления метрик
- `displayMetricsHistory()` - показать историю метрик
- `setupSurveyForm()` - настроить форму опросов ⭐ НОВОЕ
- `displaySurveyHistory()` - показать историю опросов ⭐ НОВОЕ
- `displayWeatherCorrelationAnalysis()` - показать анализ корреляции ⭐ НОВОЕ

#### `PainSurvey` ⭐ НОВОЕ
Управляет опросами о болевых ощущениях и анализом корреляции с погодой.

**Методы:**
- `addSurvey(surveyData)` - добавить опрос
- `removeSurvey(id)` - удалить опрос
- `getSurveysForDate(date)` - получить опросы за день
- `getSurveysForPeriod(startDate, endDate)` - получить опросы за период
- `getPainStatistics(painType)` - получить статистику по типу боли
- `getWeatherCorrelation(painType)` - получить корреляцию с погодой (Пирсон)
- `getTodaySurvey()` - получить опрос за сегодня
- `hasCompletedTodaySurvey()` - проверить, прошёл ли опрос сегодня

## Темы оформления

Приложение поддерживает две темы:

- **Светлая тема** (по умолчанию): белый фон, тёмный текст
- **Тёмная тема**: тёмный фон, светлый текст

Выбор темы сохраняется в LocalStorage и применяется при загрузке.

## Интеграция API

### OpenWeatherMap API
Используется для получения данных о погоде, которые могут влиять на показатели здоровья:

```javascript
const weatherData = await healthAPI.fetchWeatherData(latitude, longitude);
```

### JSONPlaceholder API
Используется для получения рекомендаций по здоровью:

```javascript
const recommendations = await healthAPI.fetchHealthRecommendations();
```

## Пример использования

```javascript
// Создание объектов
const storage = new StorageManager();
const healthMetrics = new HealthMetrics();
const userProfile = new UserProfile(storage);

// Добавление метрики
const metric = healthMetrics.addMetric('heart_rate', 72, new Date(), 'После тренировки');

// Сохранение в LocalStorage
storage.save('metrics', healthMetrics.getAll());

// Получение статистики
const stats = healthMetrics.getStatistics('heart_rate');
console.log(`Средний пульс: ${stats.avg}`);
```
